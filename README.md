# MyFace
计算机设计大赛我们来了！

## 什么是MyFace？
MyFace是一款为了让人们意识到人脸安全重要性的Web网页，同时做了手机适配。当然，我们也提供了人脸识别服务和可行的几种加密方法对人脸信息进行保护！

### 1. 人脸识别
我们采用了开源的人脸识别faceapi.js

### 2. 加密算法
目前的两种加密方法：负数据加密和同态加密，在前端进行人脸数据的不可逆加密，在后端进行数据匹配达到人脸识别的效果。

满足了国际安全标准：
- 不可逆性：加密后的数据不可逆
- 不可连接性：每个人的人脸在不同的应用中不能串用
- 可撤销性：将不可变的人脸数据变的可以变化

### 3. 技术栈
本项目是一个前后端分离的web项目

#### 3.1 前端：
基础知识：HTML、CSS、JS
框架搭建：vue、vue-cli
关键人脸识别部分：faceapi.js

#### 3.2 后端：
基础知识：python、django
